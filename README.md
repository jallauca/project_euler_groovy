# Description
Solutions in Groovy to problems from Projet Euler (http://projecteuler.net)

# Compile
    ant compile

# Run Single Problem
    ./run.sh 2
    $ Problem  2: answer=XXXXXXX

# Run Range of Problems
    ./run.sh 1 11
    $ Problem  1: answer=XXXXXXX
    $ Problem  2: answer=XXXXXXX
    ...
    $ Problem 11: answer=XXXXXXX
